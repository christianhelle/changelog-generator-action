# Squad Decisions

## Active Decisions

- The initial squad is optimized for a public GitHub Action that wraps the native `chlogr` CLI.
- Early work should prioritize action contract design, binary/runtime integration, release distribution, compatibility testing, and Marketplace-grade documentation.
- The implementation stack for the wrapper is intentionally left open until the team chooses between a JavaScript/TypeScript action and a composite-style wrapper.
